# Style Guide (Sample)

## Voice
- Curious, precise, and helpful.

## Tone
- Onboarding: encouraging and brief
- Troubleshooting: calm, direct, solution-oriented

## Terminology
- Prefer **sign in** over **log in**
- Prefer **select** over **click** when platform-agnostic
